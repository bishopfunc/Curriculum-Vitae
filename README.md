## 自己PR
私の強みは、技術力を活かした課題解決力です。実務インターンや研究、ハッカソンなどを通じて、ソフトウェア開発や機械学習を用いた課題解決に取り組んできました。また、合成生物学の世界大会iGEMでは、バイオものづくりによる社会課題解決に挑戦しました。常に「世の中に役立つものをつくる」ことを意識し、真の課題を見極め、技術を駆使して実現可能かつインパクトある解決策を追求してきました。こうした経験を仕事でも活かしていきたいと考えています。

かつてハーバー・ボッシュ法が飢饉という社会課題を技術で解決したように、現代の課題も技術によって大きく改善できると私は考えています。特に機械学習をはじめとするこの数年の技術の進歩は、これまで不可能とされていたことを成し遂げてきました。私は将来、研究者、技術者かつリーダーとして技術的なイノベーションを通じて社会課題を解決したいと強く思っています。イノベーションはまず研究から始まり、そして技術者がそれを実現し、最後にリーダーがそれを社会に広めることで初めて実現します。私はこの3つの役割を担うことで、技術と社会の架け橋となりたいと考えています。
